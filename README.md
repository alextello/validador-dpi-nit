## DESCRIPCIÓN

Valída tanto el CUI como el NIT.

```
Aplica para el país de GUATEMALA
```

```
* Ejemplo:

const { cuiValido, nitValido } = require('validador-dpi-nit');

const isCUIValid = cuiValido('2660507980909');
// output --> false

const isNITValid = nitValido('85686988');
// output --> false

```
